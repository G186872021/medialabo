「Webコンテンツ作成」の成果

本テーマの学習目的

このテーマでは、HTML, CSS, JavaScript の基本を学び，これらを組み合わせて Web ページを構成する方法を学びます。

先行する専門科目で学習した情報工学の基礎知識を応用した実験を通じて実践的な能力を修得することが目的です。到達目標として次の4つがあります。

(1) 手順書を読み、実験を行い、レポートやプレゼンテーションにより報告するという実験の一連の流れを実践できる

（2）演習実施に必要なコミュニケーション能力の重要性を理解し、実験を通じて実践できる

（3）プログラムによるロボットの制御ができる

（4）HTML、JavaScriptなどによる Webページ作成ができる。

なお、本授業は、リーダーシップ能力、課題に対する解決能力育成に寄与する科目です。

プロジェクト課題について

プロジェクト課題では都市の名前を入力し、そのデータを表示させるプログラムを作成しました。

まず、HTMLを作成し、テーマを選びました。自分は世界の天気をテーマに作成しました。

第2回の講義ではHTMLにCSSを追加しました。

第3回の講義でオブジェクトからデータを取り出し、データの一部をコンソールに表示しました。

第4回の講義では前回の課題で作った JavaScript プログラムを改良してページ上にデータを表示しましょうしました。

第5回の講義では検索欄を設置し、検索用の input 要素やラベル、ボタンを設置しました。

第6回の講義ではaxiosを使い、通信処理を組み込み、URLからデータを取り出しました。

第6回までの講義を終え、プロジェクト課題が完成まで持って行けず、中途半端な形での提出になりました。

今回の反省点は講義外での自学習の頻度があまりにも少なかったことです。

この反省を生かし、次のロボット制作課題では全て期限内に完成させられるようにしたいと思います。

講義を終えて

工夫点 index.htmlの工夫点はそれぞれの講義に分割し、課題や練習などに無駄な差別を用いずまとまったページに仕上げた。 project.htmlの工夫点は表をできるだけ見やすくするために無駄な装飾はせず、シンプルに仕上げた。 苦心点 index.htmlとproject.htmlに共通して言えることは、指定の課題などの掲示以外に縛りがなく、 自由なところがどのようなプログラムをどう組めばいいのかがわからなくなったり、雑な仕上がりになる原因でした。
